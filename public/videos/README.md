# Hero Background Video

## 视频要求

请将您的 AI 生成视频放置在此目录，并命名为 `hero-bg.mp4`

### 推荐规格：
- **分辨率**: 1920x1080 (1080p) 或更高
- **格式**: MP4 (H.264 编码)
- **时长**: 10-30 秒（会自动循环播放）
- **文件大小**: 建议 < 10MB（优化加载速度）
- **内容**: AI 视频生成相关的视觉效果，如：
  - 抽象的 AI 艺术动画
  - 科技感的粒子效果
  - 流动的数据可视化
  - 未来感的场景

### 视频优化建议：
```bash
# 使用 ffmpeg 压缩视频（如果需要）
ffmpeg -i input.mp4 -vcodec h264 -crf 28 -preset slow hero-bg.mp4
```

### 临时方案：
如果暂时没有视频，可以：
1. 使用 Seedance 2.0 生成一个科技感的视频
2. 或从 https://seedanceapi.org 下载示例视频
3. 或使用纯色/渐变背景作为临时替代
